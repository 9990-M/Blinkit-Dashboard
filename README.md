Project Title: Blinkit Sales Performance Dashboard

Description:
An interactive Power BI dashboard analyzing ~1,000–10,000 records of Blinkit sales, outlet, and product data (sourced from Kaggle) to uncover customer preferences, outlet performance, and category-level revenue trends. The raw dataset was cleaned and transformed using Power Query, then modeled with custom DAX measures to calculate KPIs such as total sales, average ratings, and item-level performance. The final dashboard features dynamic slicers and filters, enabling stakeholders to explore sales patterns by outlet and product category in real time.

Tech Stack:
Power BI – Built the interactive dashboard, visuals, and KPI cards for sales and outlet performance.
Power Query – Cleaned and transformed the raw Kaggle dataset (handled missing values, formatting, data types) before loading it into the model.
DAX – Wrote custom measures for KPIs like total sales, average rating, and item-level performance.
Excel – Used for initial data inspection and quick validation checks before importing into Power BI.

Highlights:
End-to-end workflow: raw data → Power Query cleaning → DAX-modeled KPIs → interactive dashboard
